# Coffee-Espresso

A modern and responsive coffee shop website built with HTML, CSS and JavaScript.

# Features


Responsive navigation with mobile hamburger menu

Home section with hero content

About Us section

Menu section with 10 items

Products section with ratings and pricing

Customer Reviews section

Contact section with Google Maps and contact form

Blogs section with recent posts

Footer with social media links


# Technologies Used


HTML5

CSS3

JavaScript (Vanilla)

Font Awesome - Icons

Google Maps Embed API
